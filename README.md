# MIT2024QHackthon
Solution for 2024 MIT Hackathon.


# Install the required package

In windows:

```console
py -m pip install -r requirements.txt 
```

In Mac, Linux or Ubuntu:

```console
pip install -r requirements.txt 
```

# Run the main function


In windows:

```console
py main.py
```

In Mac, Linux or Ubuntu:

```console
python3 main.py
```
